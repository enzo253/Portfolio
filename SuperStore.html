📊 SuperStore Dashboard

An interactive dashboard visualizing sales, profits, and customer segmentation for a fictional SuperStore using Plotly Dash.

🛠️ Tech Stack

  • Python, Pandas, NumPy for data wrangling
  • Dash & Plotly for interactive dashboard components
  • Tailwind CSS for styling integration
  • HTMX used for lightweight interactivity

⚙️ Process & Methodology

I started by cleaning and restructuring the sales dataset to normalize region-based performance metrics. After establishing a clean base, I used EDA to identify key drivers of profit. With Dash, I built interactive filters by region, category, and date to enable dynamic insights. Performance was optimized using callback functions and memoization to ensure fluid UX.

I also integrated light responsiveness so the dashboard performs well across devices, and connected it to a JSON config to allow plug-and-play with new datasets.

📈 Outcomes

  • Gained deeper understanding of Dash layouts & callbacks
  • Improved loading speed by 30% via efficient state management
  • Users can now identify top-performing products and segments instantly
